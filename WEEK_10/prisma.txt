Installing Prisma

1- Initialize an empty Node.js project 
   npm init -y

2- Add dependancies 
   npm install prisma typescript ts-node @types/node --save-dev

1. Initialize typescript:-
   npx tsc --init

1. Initialize a fresh prisma project
   npx prisma init


2. Generate Prisma Migrations
   npx prisma migrate dev --name Intialize the schema
   // creates migrations - migrations.sql
